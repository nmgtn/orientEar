orientEar

Ear Trainer - tech project for Tonmeister course

Download this project as a .zip file Download this project as a tar.gz file

orientEar will be a browser-based ear trainer, taking advantage of HTML5 and the Web Audio API, building upon the work done on BeaqleJS. It is being developed by Andy Normington as a Technical Project (final technical assessment) for the Tonmeister course at the Institute of Sound Recording, University of Surrey.

The project is still in early development - a demo will be available soon.
