orientEar

Ear Trainer - tech project for Tonmeister course

View on GitHub Download .zip Download .tar.gz

orientEar will be a browser-based ear trainer, taking advantage of HTML5 and the Web Audio API, building upon the work done on BeaqleJS. It is being developed by me, Andy Normington, and began life as part of my Technical Project (final technical assessment) for the Tonmeister course at the Institute of Sound Recording, University of Surrey. After graduation, I will continue development and hope to get all the core functionality running properly before too long.

The project is still in early development - a demo will be available soon. If you'd like to hear more, get in touch on GitHub or andynormington.co.uk.
